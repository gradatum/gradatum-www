Gradatum is a family of self-hosted, embedded building blocks for multi-agent AI systems. Each module solves one foundational need — memory, retrieval, identity, coordination — with the same discipline: zero external services, Rust-first, MCP-native.

Why an umbrella

One discipline.
Many primitives.

Modern agent stacks are stitched from a dozen vendor SaaS, each with its own auth, its own latency, its own privacy story. Gradatum builds the opposite — a coherent family of local-first primitives, each shippable on a single LXC, each composable via the Model Context Protocol, each owned end-to-end.
